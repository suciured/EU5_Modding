EU5 T-RANK TEMPLATE
Version: Authority-Based System
Status: Baseline (Converted from R-Rank)
Purpose: Canon-safe reference for continuation and implementation

==================================================
OVERVIEW
==================================================

T-rank is a hidden structural ranking system derived from Net Authority (NA).
It represents how authority is organized within a state.

- T-rank is never shown directly to the player
- T-rank is never manually set
- T-rank is always derived from Net Authority math
- Government type and subject type are orthogonal to T-rank
- Decimal subranks (.1–.6) correspond directly to UI slots (1–6)

==================================================
FORMAT NOTES
==================================================

- Only structural identifiers are used (numeric subranks)
- No flavor names are used
- Any undefined data is marked as TBD
- Country-rank distribution is bottom-heavy
- Only two Empire tiers exist (T14–T15)
- Upgrade constraints default to TBD
- Exceptions:
  - T10 → T11 requires Independence
  - T15 is the maximum rank
