============================================================
CORE TERMINOLOGY
============================================================
CA/LA/TR LEGACY (DEPRECATED IN MOD)
---------------------------------
The original Canon KB terminology uses:
- CA = Central Authority
- LA = Local Authority
- NA = Net Authority (CA − LA)
- TR = Title Rank (hidden tier derived from TA)
- T-rank = UI node tree (Tn.m)

This MOD replaces the authority math with the V-system below.

MOD CANON TERMINOLOGY (V-SYSTEM)
--------------------------------
- Central Base  = V3.7 (country-scoped, derived)
- Local Base    = V3.8 (country-scoped, derived)
- Net Authority = V4.3 (per-tick flow)
- Title Authority (TA) = v4_ta (persistent accumulator)
- T-rank (integer) = v5_t_rank, T0–T15 derived from TA thresholds (V5)
- T-rank UI grid = Tn.m slots (UI structure only; does not set integer rank)

INDEPENDENCE GATE (MOD CANON)
-----------------------------
- T11+ requires Independence.
- If NOT independent: clamp integer T-rank to at most T10.
- Subjects use the same rules but accumulate TA from halved central/local contributions.
